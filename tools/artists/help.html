作品编辑器使用帮助

功能简介

本编辑器允许您通过可视化界面快速编辑成员信息与作品内容，支持动态添加多项作品及媒体文件，最终生成可下载的 HTML 文件。

使用步骤

  1. 填写顶部的标题、职位、成员名字与成员描述信息。
  2. 点击“添加作品”按钮添加新的作品项。
  3. 为每个作品填写标题和详情。
  4. 点击“添加媒体”按钮为该作品添加图片链接或 Bilibili 视频嵌入链接。
  5. 重复以上步骤可添加多个作品和媒体。
  6. 点击“更新预览”按钮以查看右侧的实时预览效果。
  7. 确认无误后，点击“下载 HTML 文件”按钮生成最终页面。

媒体链接说明

1. 获取图片的 URL 地址

  • 从网络获取图片 URL：
    • 在浏览器中找到所需的在线图片。
    • 右键点击该图片，选择“复制图片地址”或类似选项。
    • 将复制的链接粘贴到编辑器的媒体链接输入框中。
  • 从本地上传图片并获取 URL：
    • 使用图床服务（如 SM.MS）上传本地图片。
    • 在网站上点击“Browse...”按钮，选择并上传图片。
    • 上传成功后，网站会生成该图片的 URL 地址，复制该链接并粘贴到编辑器的媒体链接输入框中。

2. 获取 Bilibili 视频的嵌入链接

  • 打开 Bilibili 网站，找到想要嵌入的视频。
  • 点击视频下方的“分享”按钮。
  • 在弹出的选项中，点击“嵌入代码”，并复制提供的 <iframe> 代码。
  • 从复制的 <iframe> 代码中，提取 src 属性的 URL（例如，https://player.bilibili.com/player.html?bvid=XXXX）。
  • 将提取的链接粘贴到编辑器的媒体链接输入框中。

生成的 HTML 文件

下载的 HTML 文件将自动包含引用本地同级目录的 style.css 和 script.js 文件，以确保样式和交互行为一致。
